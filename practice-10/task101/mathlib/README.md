# MathUtils - Библиотека математических функций

Библиотека, содержащая базовые математические функции: факториал, числа Фибоначчи, НОД, НОК и проверку на простоту.

## Возможности

- `factorial(n)` - вычисление факториала числа n (n!)
- `fibonacci(n)` - вычисление n-го числа Фибоначчи
- `gcd(a, b)` - нахождение наибольшего общего делителя
- `lcm(a, b)` - нахождение наименьшего общего кратного
- `is_prime(n)` - проверка числа на простоту

## Сборка

Проект поддерживает сборку как статической, так и динамической библиотеки для Linux и Windows.

### Сборка с помощью Make

```bash
# Сборка всего (статическая и динамическая библиотеки)
make all

# Только статическая библиотека
make static
./program_static

# Только динамическая библиотека
make shared
./program_shared

# Очистка
make clean

# Информация о собранных файлах
make info
