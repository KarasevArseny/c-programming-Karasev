# Кроссплатформенный Makefile для сборки библиотек

Универсальный Makefile для сборки статических и динамических библиотек в Linux и Windows (MSYS2, MinGW).

## Возможности Makefile

- ✅ Автоматическое определение операционной системы
- ✅ Сборка статических библиотек (.a)
- ✅ Сборка динамических библиотек (.so в Linux, .dll в Windows)
- ✅ Сборка тестовых программ с обеими версиями
- ✅ Информация о собранных файлах
- ✅ Поддержка очистки
- ✅ Кроссплатформенность

## Структура проекта

project/
├── mathutils.h # Заголовок математической библиотеки
├── mathutils.c # Реализация математических функций
├── strutils.h # Заголовок строковой библиотеки
├── strutils.c # Реализация строковых функций
├── main.c # Тестовая программа
├── Makefile # Универсальный Makefile
└── README.md # Документация

## Использование

### Сборка

```bash
# Сборка всего (статическая и динамическая версии)
make

# Только статические библиотеки
make static

# Только динамические библиотеки
make shared

# Информация о собранных файлах
make info

# Запуск тестовой программы
make test

# Проверка утечек памяти (Linux)
make valgrind

# Полная проверка (сборка + информация + тесты)
make check

# Очистка
make clean

# Справка
make help
