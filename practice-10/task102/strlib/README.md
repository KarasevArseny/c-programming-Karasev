# StrUtils - Библиотека для работы со строками

Библиотека, содержащая функции для обработки строк: реверс, изменение регистра, обрезка пробелов и подсчёт слов.

## Возможности

- `str_reverse(s)` - реверс строки (возвращает новую строку)
- `str_to_upper(s)` - преобразование в верхний регистр (возвращает новую строку)
- `str_to_lower(s)` - преобразование в нижний регистр (возвращает новую строку)
- `str_trim(s)` - удаление пробелов по краям строки (возвращает новую строку)
- `str_word_count(s)` - подсчёт количества слов в строке

## Важно!

Все функции, возвращающие `char*`, выделяют память через `malloc`. 
Вызывающий код **обязан** освободить эту память через `free`.

## Сборка

Проект поддерживает сборку как статической, так и динамической библиотеки для Linux и Windows.

### Сборка с помощью Make

```bash
# Сборка всего (статическая и динамическая библиотеки)
make all

# Только статическая библиотека
make static
./program_static

# Только динамическая библиотека
make shared
./program_shared

# Очистка
make clean

# Информация о собранных файлах
make info

# Проверка утечек памяти (Linux, требуется valgrind)
make test
