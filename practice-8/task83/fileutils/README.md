# FileUtils - Библиотека файловых утилит

Библиотека для работы с файловой системой: манипуляции с путями, чтение/запись файлов, копирование, перемещение и удаление.

## Структура проекта
fileutils/
├── include/ # Заголовочные файлы
│ ├── fileutils.h # Основной заголовок
│ ├── path.h # Функции для работы с путями
│ └── io.h # Функции ввода/вывода
├── src/ # Исходные файлы
│ ├── main.c # Демонстрационная программа
│ ├── path.c # Реализация функций путей
│ └── io.c # Реализация функций ввода/вывода
├── tests/ # Тесты
│ └── test_fileutils.c
├── Makefile # Сборка через GNU Make
├── meson.build # Сборка через Meson
└── README.md

## Возможности

### Работа с путями (path.h)
- `path_join` - объединение частей пути
- `path_dirname` - получение директории пути
- `path_basename` - получение имени файла
- `path_extension` - получение расширения файла
- `path_without_extension` - путь без расширения
- `path_normalize` - нормализация пути
- `path_is_absolute` - проверка на абсолютный путь
- `path_exists` - проверка существования
- `path_is_file` - проверка на файл
- `path_is_directory` - проверка на директорию

### Функции ввода/вывода (io.h)
- `file_read_all` - чтение всего файла
- `file_write_all` - запись в файл
- `file_append` - дописывание в файл
- `file_read_lines` - построчное чтение
- `file_copy` - копирование файла
- `file_move` - перемещение файла
- `file_delete` - удаление файла
- `file_size` - размер файла
- `file_exists` - проверка существования
