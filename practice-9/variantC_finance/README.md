# Система учёта финансов

Консольное приложение для учёта доходов и расходов с возможностью ведения нескольких счетов, категоризации транзакций и формирования отчётов.

## Возможности

- ✅ Управление несколькими счетами
- ✅ Категоризация транзакций
- ✅ Добавление, редактирование, удаление транзакций
- ✅ Фильтрация и сортировка транзакций
- ✅ Статистика за период
- ✅ Отчёты по категориям
- ✅ Экспорт в CSV
- ✅ Сохранение данных в бинарном файле

## Требования

- Компилятор C (gcc/clang)
- Make ИЛИ Meson
- Стандарт C11

## Сборка

### С использованием Make

```bash
# Сборка debug версии
make

# Сборка release версии
make release

# Запуск программы
make run

# Запуск тестов
make test

# Очистка
make clean

#### С использованием Meson

# Настройка сборки
meson setup builddir

# Сборка
meson compile -C builddir

# Запуск
./builddir/finance_manager

# Запуск тестов
meson test -C builddir

# Release сборка
meson setup builddir_release --buildtype=release
meson compile -C builddir_release
